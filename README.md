# VibeDev

Grundstruktur för `vibedev.se` byggd med Next.js, TypeScript och Tailwind CSS.

## Struktur

- `app/` innehåller routes via App Router
- `components/` innehåller återanvändbara UI- och layoutkomponenter
- `content/` innehåller strukturerad innehållsdata
- `lib/` innehåller metadata och hjälpfunktioner

## Start

```bash
npm install
npm run dev
```
